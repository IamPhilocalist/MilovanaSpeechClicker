Speech Clicker
by Philocalist

This is a chrome extension created for the sole purpose of clicking buttons within milovana.com webteases using speech recognition.

Because this chrome extension uses a chrome api for speech recognition you should use it inside Google Chrome or it probably won't work.

After you click Start it will ask you for your permission to use the microphone on your computer. You should choose "Allow this time". If you click on the screen this popup window will disappear, but will continue to work.

That's it, you're ready to go. If you say the text written on the button out loud it will be clicked. After you close the page the speech recognition goes away as indicated by the microphone sign on the top bar.

Feel free to use this tool as you wish.
